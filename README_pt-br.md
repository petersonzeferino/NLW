# NLW#1 Booster

## 1. Objetivo
Projeto para cadastro de pontos de coleta de resíduos

## 2. A solução
Construção de aplicativos web e mobile, utilizando Typescript com base em NodeJs, React e React Native

## 3. Pré Requisitos
Instalar o NodeJs e as dependencias
```bash
npm install
```

## 4. Execução
- **backend** 
```bash
npm run dev
```

- **frontend e mobile:**
```bash
npm start
```
